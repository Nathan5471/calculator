# Calculator

## Description

This is a web based calculator that has three different difficuly settings: easy, medium, and hard. Easy mode is just a normal calculator. Medium shuffles around the keys every time you press one. Hard shuffles the keys and then will make them dissapear after two seconds.

## Technologies

This project was made with React, TypeScript, and TailwindCSS. It is currently deployed with Vercel at <https://calculator-y7eb8wuf4-nathan5471s-projects.vercel.app/>
